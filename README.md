# Customer Support Simulator

Welcome to the Customer Support Simulator.

This repository contains realistic customer support scenarios that simulate real conversations between customers and support agents.

The purpose of this project is to demonstrate communication skills, problem-solving, empathy, conflict resolution, and professional customer handling.
